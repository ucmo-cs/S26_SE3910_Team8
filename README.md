# Team 8 Scheduler Project

This repository contains:

- Frontend: React + Vite app in the project root
- Backend: Spring Boot API in `backend`

## Frontend

Install and run from project root:

```bash
npm install
npm run dev
```

If your backend runs on a different host/port, set:

```bash
VITE_API_BASE_URL=http://localhost:8080/api
```

## Backend

See `backend/README.md` for full setup steps.

Quick start:

```bash
cd backend
mvn spring-boot:run
```
